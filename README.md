## 📌 About

This repository tracks my Data Structures and Algorithms preparation. Problems are organized **by platform**, with solutions written in multiple languages depending on the problem or platform requirements. The goal is consistent daily/weekly practice, deep understanding of core concepts, and interview readiness.

## 📂 Repository Structure

```
dsa-preparation/
├── LeetCode/
│   ├── Arrays/
│   ├── Strings/
│   ├── LinkedList/
│   ├── Trees/
│   ├── Graphs/
│   ├── DP/
│   └── ...
├── Codeforces/
│   ├── Div2/
│   ├── Div3/
│   └── ...
├── HackerRank/
├── GeeksforGeeks/
├── InterviewBit/
├── Notes/
│   └── topic-wise cheat sheets
└── README.md
```

Each problem folder typically contains:
- `problem.md` — problem statement / link / notes
- Solution file(s) (`.cpp`, `.java`, `.py`, etc.)
- Time & Space complexity in comments or notes
